UIView+Rotation3D
==========

3D Rotation for UIView

## Features

- UIView category
- Show / Hide


## How To Use


### 1. Initialize
```
view.rotation3DAnimationDuration = duration;
view.rotation3DAnimationDelay = delay;
```

### 2. Action
```
[view showRotation3D];
```
OR
```
[view hideRotation3D];
```
